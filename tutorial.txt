Backend-
1- criar pasta backend dentro do projeto
2- no powershell abrir a pasta e digitar o comando ->    yarn init -y
3- dentro da pasta backend ainda no terminal do vscode -> yarn add express
4- criar arquivo index.js na pasta backend
5- para executar no servidor o index.js -> node index.js

para o node atualizar automaticamente -> yarn add nodemon -D
após instalar o nodemon, utilizar -> yarn nodemon index.js
